Frequently Asked Questions

Understanding the transition to an integrated project management ecosystem.

Need direct assistance?

Our Institutional Support team is available for departmental onboarding.

How is data ownership handled?

expand_more

The institution maintains 100% data sovereignty. All research artifacts are stored on encrypted servers with custom retention policies dictated by your university's governance.

Can we integrate with existing LMS platforms?

expand_more

Yes, Scholarly Monolith supports LTI and REST API integrations for common platforms like Canvas, Moodle, and Blackboard.

Is there a limit on project collaborators?

expand_more

Institutional licenses allow for unlimited internal collaborators. External guest access can be configured per-project by the primary investigator.
